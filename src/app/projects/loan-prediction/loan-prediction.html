Loan Prediction Model

Project Description

A class Kaggle competition on an industry-scale dataset, where the goal was to predict whether a borrower would repay their mortgage. The input data was constructed of structured but uncleaned csv rows with both categorical and numerical columns such as loan amount, zip code, and race. I tested the problem across the model space (logistic regression, decision trees, k nearest neighbor (KNN)) and training strategies (bagging, boosting, k-fold cross validation), pairing each with different preprocessing and data-pruning approaches. Gradient Boosted Decision Trees had the highest accuracy out of all the approaches I tried, but the more interesting problem was bias: race wasn't a reliable field, yet zip code acted as a proxy for it due to historical redlining. Part of the work was identifying and correcting for that rather than pushing for more advanced models.

Summary

  • Predicted mortgage loan repayment using a Gradient Boosted Decision Tree, outperforming simpler baselines
  • Evaluated multiple model families and training strategies (bagging, boosting, k-fold CV) to find the best-performing approach
  • Identified and mitigated demographic bias in the data, including zip code acting as a proxy for race due to redlining
